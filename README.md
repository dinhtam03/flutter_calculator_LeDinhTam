Dự án này là một ứng dụng máy tính đơn giản (Simple Mobile Calculator) được xây dựng bằng Flutter, nằm trong bài thực hành LAB 2 – Simple Mobile Calculator.
Ứng dụng được thiết kế dựa trên Figma mẫu của môn học, với giao diện trực quan và đầy đủ các chức năng của một máy tính cơ bản.

  Thông qua dự án này, sinh viên luyện tập các kỹ năng:
      - Xây dựng UI bằng Flutter
      - Quản lý trạng thái (State Management)
      - Xử lý sự kiện và nhập dữ liệu
      - Thực hiện các phép toán số học
      - Xử lý lỗi như chia cho 0
      - Tổ chức code khoa học, dễ bảo trì
      - Chuyển bản thiết kế Figma thành giao diện Flutter hoàn chỉnh
